## General info
This project, Evening Stroll, is a code prototype created for my Working with Data and Code class. Evening Stroll is an ASCII converter that transforms any video into a text-based art animation, turning each frame into a sequence of ASCII characters.
	
## Technologies and Requirements
Evening Stroll is created with:
* Python 3.13.5
* Visual Studio Code
* Python Libraries:
  * Open CV 
  * Pillow 
  * Time

## Features
* Customisable Output
  * Users can adjust the ASCII character set, resolution, and frame rate to create different visual effects.

* Video-to-Text Conversion
  * Transforms any video into a flowing ASCII art animation in real time.
